1. Wymień wszystkie operatory matematyczne w Javascript.

Operatami matematycznymi są:
- Addition (+),
  Operator (+) sumuje liczby. Dodaje dwie liczby, zwraca wynik dodawania tych dwóch liczb. Dodając numer (jaki numer?) i ciąg (łańcuch (jaki łańcuch?)) zwróci ciąg (jaki ciąg?).
  Na przykład:
  Dodawanie dwóch liczb:
  const x = 4 + 4; -> wynik wynosi 8

  Dodawanie numeru z ciągiem:
  const x = “4” + 4; -> wynosi 44 (nie, zwraca “44”)
  const x = “foo” + 4 -> foo4 (Co tu zwróći?)
  const x = “foo” + false -> foofalse (Co tu zwróci?)

- Subtraction (-),
  Operator (-) odejmuje liczby. Odejmuje dwie liczby, zwróci wynik odejmowania dwóch liczb. Odejmując ciąg i numer zwróci NaN (oznacza “nic”, puste - NIEPRAWDA!!!).
  Przykład;
  const x = “hello” - 4; -> NaN
  const x = 5 - 4; -> 1
  const x = 4 - true; -> 3 (Dlaczego??)
  const x = 4 - false; -> 4 (Dlaczego?)
  const x = 4 - 5; -> -1

- Multiplication (*),
  Operator zwraca wynik mnożenia. Mnoży dwie liczby, zwraca wynik ich mnożenia.
  Przykład:
  const x = 2 * 3; -> 6
  “foo” * 2 -> NaN (Co to znaczy?)
  Infinity * Infinity -> Infinity
  Infinity * 0;-> NaN

- Division (/),
  Zwraca iloraz operandów, gdzie po lewej stronie jest dzielną, a po prawej dzielnikiem.

  1.0 / 2.0 -> zwraca 0.5
  2 / 0 -> Infinity

  Modulo (%),
  Zwraca resztę dzielenia liczby całkowitej.

  12 % 5 -> 2
  -1 % 2 -> -1
  Nan % 2 -> NaN
  2 % 3 -> 2
  4 % 2 -> 0

- Increment (++),
  Operator inkrementacji zwiększa o wartość operandu i zwraca taką wartość.
  Przyrost (x++) zwracana wartość nie będzie powiększona np:
  const x = 3;
  y = x++ -> x =4, y = 3
  Przedrostek (++x) zwracana wartość będzie powiększona
  const x = 3;
  y = ++x; -> y = 4, x = 4

- Decrement (--).
  Operator dekrementacji zmniejsz o 1 wartość operandu i zwraca taką wartość.
  Przyrostek(x--) zwracana wartość nie będzie pomniejszona
  const x = 3;
  y = x--; -> y=3, x=2
  Przedrostek (--x) zwracana wartość będzie pomniejszona.
  const x = 3;
  y = --x; -> x=2, y=2
